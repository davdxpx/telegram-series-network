# Contributing to Telegram Series Network

Thank you for your interest in contributing to TSN Bot! We welcome contributions from the community.

## Getting Started

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally.
3.  **Create a new branch** for your feature or bugfix:
    ```bash
    git checkout -b feature/my-new-feature
    ```
4.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

## Development Standards

*   **Code Style**: We use `black` for formatting and `isort` for import sorting. Please ensure your code follows these standards.
*   **Type Hinting**: All new code must be fully type-hinted (Python 3.12+ syntax).
*   **Testing**: If you add a new feature, please add relevant tests (if applicable) or ensure existing functionality is not broken.

## Pull Request Process

1.  Ensure your code passes all linting and local tests.
2.  Update the `README.md` if you change any user-facing functionality.
3.  Submit a Pull Request with a clear description of your changes.

## Code of Conduct

Please be respectful and inclusive in all interactions. Harassment or abuse will not be tolerated.
